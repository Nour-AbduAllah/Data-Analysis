Sackoverflow developer DB

This project done in 3 main steps:
	1-Data gathered and cleaned.
	2-Data Explored
	3-Questiones Asked and Answered

We did analysis for this DB to answer some questions like:
	-What is the most popular programming language in 2020?

	-Which languages are the most people interested to learn over the next year?

	-Which are the most loved languages i.e. a high precentage of people who have used the
		language want to continue learning and using it over the next year?

	-In which countries do developers work the highest number of hours per week? 

	-How important id it to start young to build a career in programming?



Tools used:
	-Python
	-Numpy
	-Pandas
	-Matplotlib
	-Seaborn
	-EDA.
