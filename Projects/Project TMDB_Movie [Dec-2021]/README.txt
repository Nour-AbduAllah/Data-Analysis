The Movie Data Base

This project done in 4 main steps:
	1-Intro.
	2-Data Wrangling
	3-EDA
	4-Conclusions

We did analysis for this DB to answer some questions like:
	-Which genres are most produced?

	-Top 20 directors who directes most movies?

	-Which year&month has the most releases?

	-which year&month has the higher revenue?

	-What is the average of budget, revenue, profit for each year?

	-What is the type of correlation between Revenue and Popularity, Rating, Runtime?



Tools used:
	-Jupyter notebook
	-Python
	-Numpy
	-Pandas
	-Matplotlib
	-Seaborn
	-EDA.
